Lógica de Programação - Portugol

O Portugol é uma pseudolinguagem utilizada como ferramenta de ensino para introduzir conceitos de lógica de programação. Ele não é uma linguagem de programação real, mas sim um modo de representar algoritmos de forma estruturada e compreensível para quem está aprendendo a programar.

Sintaxe:
A sintaxe do Portugol é simples e segue uma estrutura próxima à linguagem natural, o que facilita o entendimento para iniciantes.

Além disso, é composto por elementos como:

1. Algoritmo: É o início do código e representa o nome do algoritmo que será desenvolvido.

2. Declaração de Variáveis: Pode-se definir variáveis e seus tipos (inteiro, real, caractere etc.).

3. Estruturas de Controle: São usadas para controlar o fluxo do programa, como estruturas condicionais (se...então...senão) e laços de repetição (para, enquanto).

Entrada e Saída de Dados: Possui comandos para entrada (leia) e saída (escreva) de dados.

Funcionamento:
O Portugol é utilizado principalmente para explicar como os algoritmos funcionam. A ideia é representar a lógica por trás de um programa sem se preocupar com a sintaxe de uma linguagem de programação específica.
Por exemplo, para calcular a média de dois números, pode-se criar um algoritmo que siga os passos: 

 
    algoritmo media    
        var
        nota1, nota2, media: real
            
    inicio
        escreva("Digite a primeira nota: ")
        leia(nota1)
            
        escreva("Digite a segunda nota: ")
        leia(nota2)
            
        media <- (nota1 + nota2) / 2
            
        escreva("A média é: ", media)
    fim
